Ссылки по теме HTML5 и Canvas

Документация

  • HTML5 Game Development Tutorial — по этой статье я научился рендерить и двигать картинки, выводить текст и проигрывать звук. Ещё там есть примеры реализации простой физики и поворота картинок.
  • JavaScript Gaming: Часть 1. Box2d и основы Физики — статья о том, как сделать прототип Angry Birds на box2djs. Именно с неё началось мое знакомство с box2d.
  • Archive for the ‘Box2D’ Category — интересные статьи по флешевому box2.
  • Box2D v2.0.2 Руководство — перевод документации.
  • (дочитать примеры) MDN. Обучение canvas — очень толковый туториал от Mozilla (можно переключить язык на русский).
  • (поковырять код) История Лошарика — как ребята сделали игру типа RapidRoll.
  • 3d движок на js + canvas — с экспортом модели из 3ds max.

Движки

  • box2dweb — a port of Box2DFlash 2.1a to JavaScript.
  • LimeJS — is a HTML5 game framework. Есть документация и примеры.
  • Chipmunk — js порт популярного движка.
  • Three.js — 3d движок на Javascript.
